 - Download the compressed, production jQuery ... 누르면 js 파일 읽어준다 그걸 복사해서 붙여넣기해서 적당히 이용
 - `<script>https://code.jquery.com/jquery-3.6.0.min.js</script>를 적어준다

```html
<script>
	window.onload = function(){}
    // 를 이제는

```
